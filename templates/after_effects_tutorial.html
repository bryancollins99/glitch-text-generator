Animate Glitch Text in After Effects

Learn to create dynamic glitch text animations in Adobe After Effects. Want instant glitch text? Try our online glitch text generator!

Method 1: Digital Glitch Animation

  1. Create Text Layer

    Create a new composition (1920x1080, 30fps) and add your text layer.

  2. Duplicate and Offset

    Create three copies of your text. Apply different colors (RGB) to each layer.

    Position Keyframes:
Layer 1: [0,0] → [+5,0]
Layer 2: [0,0] → [-5,0]
Layer 3: [0,0] → [0,+3]

  3. Add Expressions

    Apply this expression to position property:

    wiggle(random(2,10), random(5,20))

  4. Time Displacement

    Add Time Displacement effect and animate the evolution property.

Method 2: Glitch Transition Effect

  1. Prepare Text Layers

    Create two text layers for your transition.

  2. Add Effects

    • Venetian Blinds
    • Wave Warp
    • Channel Shift

  3. Animation Setup

    Keyframe these properties:

    Wave Warp: Wave Height
Channel Shift: Shift Channels
Venetian Blinds: Transition

  4. Fine-tuning

    Add RGB separation and chromatic aberration for extra effect.
