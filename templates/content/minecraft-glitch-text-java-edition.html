It's one of the most visually striking formatting options available in the game.

This creates the illusion of text that's constantly shifting and impossible to read — perfect for building suspense or mystery in your world.

Glitch Text in Java Edition vs Bedrock Edition

Important Notes and Limitations

Before you go all-in on glitch text, there are some practical considerations to keep in mind.

Many multiplayer servers disable or filter § codes in chat to prevent formatting abuse and spam. If your obfuscated text isn't working on a server, this is almost certainly why. Some anti-cheat plugins also flag § usage as a potential exploit, which could result in warnings or temporary bans.

Accessibility matters. Obfuscated text is completely inaccessible to screen readers and visually impaired players. If you're building content for a public server or shared map, provide alternative ways to access important information that's presented as glitch text.

Excessive use of obfuscated text can cause minor client-side performance issues, especially on lower-end machines. The constant character cycling requires frame-by-frame rendering updates. A few signs won't cause problems, but hundreds of obfuscated text displays in a small area might.

Always check server rules before using formatting codes in multiplayer environments. What's acceptable on one server might get you muted or banned on another.
