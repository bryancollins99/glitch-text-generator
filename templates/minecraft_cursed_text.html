Where cursed text renders inside Minecraft

Cursed text is built from Unicode combining marks stacked over ordinary letters, so it behaves like any pasted symbol. It survives in the parts of the game that accept raw Unicode and gets filtered in the parts that sanitise chat. Test the exact spot before you commit a build to it.

Frequently asked questions

Does cursed text work in Minecraft signs and books?

Yes. Signs, written books and anvil renames accept pasted Unicode, so cursed letters render. Keep intensity low on signs so the marks stay inside the character limit.

Why does my cursed text vanish in chat?

Vanilla Java chat filters most stacked combining marks, so heavy cursed text collapses to plain letters. Signs, books and plugin MOTDs keep the effect.

Can I set a cursed username?

Account usernames only allow letters, numbers and underscores. Use cursed text for plugin nicknames, display names, team prefixes and lore instead.

Is cursed text safe to paste?

It is only Unicode characters, so nothing is executed. Very high intensity can make lines hard for other players to read, which is the only real drawback.
