üíæ Font Installation Guide

Step-by-step instructions for installing glitch fonts on different operating systems.

üñ•Ô∏è Windows Installation

  1. Download the font file (usually .ttf or .otf format)
  2. Right-click the downloaded font file
  3. Select "Install" from the context menu
  4. Or copy the font file to: C:\Windows\Fonts\
  5. Restart any programs that were open to see the new font
üí° Pro Tip: For fonts you use frequently, create a "Fonts" folder in your Downloads for easy access.

üçé macOS Installation

  1. Double-click the downloaded font file
  2. Font Book will open automatically
  3. Click "Install Font" in the preview window
  4. Or drag the font file to the Font Book application
  5. The font is now available in all applications
üí° Pro Tip: Use Font Book to organize fonts into collections for easier management.

üêß Linux Installation

  1. Create a fonts directory: mkdir -p ~/.fonts
  2. Copy font files to: ~/.fonts/
  3. Update font cache: fc-cache -fv
  4. Verify installation: fc-list | grep "FontName"
  5. Restart applications to see the new fonts
üí° Pro Tip: System-wide installation: copy to /usr/share/fonts/ (requires sudo).

üåê Web/CSS Installation

  1. Upload font files to your web server
  2. Add CSS @font-face declaration:
@font-face {
    font-family: 'GlitchFont';
    src: url('fonts/glitch-font.woff2') format('woff2'),
         url('fonts/glitch-font.woff') format('woff');
    font-weight: normal;
    font-style: normal;
}

.glitch-text {
    font-family: 'GlitchFont', sans-serif;
}
üí° Pro Tip: Use WOFF2 format for better compression and loading speed.
