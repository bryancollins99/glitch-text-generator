Create Glitch Text Effects in Photoshop

Learn how to create stunning glitch text effects in Adobe Photoshop. Need a quick solution? Try our online glitch text generator!

Method 1: RGB Split Glitch Effect

  1. Create Your Text Layer

    Start by creating a new document (1920x1080px) and add your text in white on a black background.

  2. Duplicate and Color

    Duplicate your text layer twice. Color the top layer red, middle cyan, and keep the bottom white.

  3. Offset Layers

    Use the Move tool to slightly offset the red layer to the left and the cyan layer to the right.

  4. Add Distortion

    Select each colored layer and go to Filter > Distort > Wave to add glitch-like distortion.

Method 2: Glitch Displacement

  1. Prepare Your Text

    Create your text in white on a dark background.

  2. Create Displacement Map

    Create a new layer, fill with 50% gray, add noise (Filter > Noise > Add Noise).

  3. Apply Displacement

    Go to Filter > Distort > Displace, use the noise layer as your displacement map.

  4. Color Channels

    Use the Channel Mixer to create color separation effects.
